# Shipment Dashboard
Overview :
A simple and responsive Shipment Status Dashboard built to track logistics shipments.
This project was developed as part of a **technical assessment**.
# Techstack used 
- React Js using Vite
- Typescript for typechecking,
- Tailwind Css for styling
## Features
- Display shipment details including tracking number, status, and delivery information
- Status-based filtering using tabs
- Color-coded shipment status indicators
- Expandable “View Details” section for additional shipment information
- Responsive layout for desktop and mobile screens
- Simulated API call with loading and error states

## How to Run the Project Locally
```bash
git clone https://github.com/hm-aditya/shipment-dashboard.git
cd shipment-dashboard
npm install
npm run dev
```
The app will be available at:
http://localhost:5173
